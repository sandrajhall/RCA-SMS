Development Environment Setup

Install .NET 10 Framework
Install Visual Studio 2026 (required for .NET 10)
Install MS SQL Server (latest Developer Standard Edition) - You may need to get SOM IT to install this for you.  If they install it, be sure to have them log in to the "." server and add your account as a "sysadmin".
Install SQL Server Management Studio (SSMS)

Get the latest copy of the database from the Osprey computer (osprey.bioinf.unc.edu) in the DB Backups folder.
In SSMS, log in to the "." server.
Create a new database called RCA-SMS and restore the backup file.  (See Common Tasks document for restore instructions.)
***When local dev DB is setup, add instructions to connect there to use the existing DB.***

Get the latest copy of the codebase from Gitlab under lims\RCA_StudyManagementSystem.
Navigate to where you want to save the code and clone the repo.
> git clone git@sc.unc.edu:lbg_lims/rca-sms.git

Open the solution file (RCA_StudyManagementSystem.slnx) in Visual Studio.

Make sure the top project in the list is the startup project (RCA_StudyManagementSystem).  It should be bold.  If not, right-click on the project and choose Set as Startup Project.

I recommend setting up a separate Chrome user profile just for development.  That way you'll have minimal interference from extensions and you can clear all data easily without deleting data you might want in your own profile.
